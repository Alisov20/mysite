Основные симантические элементы на странице:

Шапка сайта - header (верхняя часть страницы)

Подвал сайте - footer (Нижняя часть страницы)

Блок информации -article (Целостный блок информации на странице) Должен содержать заголовки h1-h6

Блок section - (Объединяет связанные между собой куски информации документа(например новости, объединенные по категориям)) Должен содержать заголовки h1-h6

Блок aside - Боковая панель Должен содержать заголовки h1-h6

Блок main - основное, не повторяющееся содержимое страницы Должен содержать заголовки h1-h6
